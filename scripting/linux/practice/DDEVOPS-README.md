# Devops guide
1. Concepts of Software Development cycle
        - Need to understand how developers work and collaborate (agile, Jira workflows)
        - What git workflow they use
        - How applications are configured (Build & Packaging tools)
        - Automated testing and test scopes
        - And generally understand what the whole software development lifecycle covers from idea to    code, all the way to releasing it to the end users!

2. OS & Linux Basics
    - So you need to know the basics of how to administer a server and install different tools on it.
    - Basic concepts of operating systems you need to understand:
        1) Shell Commands
        2) Linux file system and permissions
        3) SSH Key Management
        4) Virtualization
    - Also need to know the basics of Networking and security in order to configure the infrastructure,like:
        - Configure Firewalls to secure access
        - Understands how IP addresses, ports and DNS works
        - Load balancers
        - Proxies
        - HTTP/HTTPS
